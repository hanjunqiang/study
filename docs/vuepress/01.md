## 启动命令
`也可以使用package.json中封装的指令 npm run docs:dev`
```js
vuepress dev docs
```
## 构建命令
`该命令可以忽略,下面封装了打包和部署命令`
```js
vuepress build docs
```

## 打包和部署([问题1](#wenti1))
`执行该命令可直接打包&部署到GitHub,是因为安装配置了前提条件中插件`
```js
cnpm run deploy:build;
```
