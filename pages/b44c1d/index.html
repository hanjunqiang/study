# 打包和部署(问题1)

执行该命令可直接打包&部署到GitHub,是因为安装配置了前提条件中插件

cnpm run deploy:build;
1
前提条件
问题1

sh: gh-pages: command not found

解决方案:

  • 删除根目录依赖 (node_modules)
  • 执行命令:
    npm install (执行该步,安装vdoing等依赖)
    cnpm install gh-pages --save-dev (更新Git插件gh-pages)
  • 再次执行打包部署命令
    cnpm run deploy:build;
